Looping Through a String


A string is a sequence of characters. 
In Python you can loop through a string the same way you loop through a list. Each iteration gives you one character.

word = "Engineering Computing"
for ch in word:
    print(ch)

(a)  Write a loop that takes a word (use your own name) and prints each character on a separate line, along with its position number starting from 1.
